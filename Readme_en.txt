------------------------------------------------------------------------------------
--------------------------- School management application --------------------------

-----------------------------------------Features-----------------------------------

1) Applications management
2) Monitoring of students
3) note management
4) Planning of lessons and homework
5) printing of school report

-----------------------------------Instructions-------------- ---------------------


1) Install MySQL and its connector for C ++
2) Create a gescole database and import the data from the BD-v1.sql file
3) Import the project under Qt and configure the parameters in the database
   in the service.cpp file
